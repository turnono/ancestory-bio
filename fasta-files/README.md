# FASTA Files - Ancestral Cannabinoid Oxidocyclases

This directory contains the amino acid sequences for the three resurrected ancestral enzymes from the Wageningen University study (December 2025).

## Files

### A1A2a.fasta
**Common Ancestor** - The most ancient reconstructed cannabinoid oxidocyclase
- **Confidence Score**: 0.95
- **Specialization**: Promiscuous (30% THCA, 35% CBDA, 35% CBCA)
- **Characteristics**: Highest stability, broadest substrate specificity
- **Industrial Use**: Multi-cannabinoid production systems

### Ca.fasta
**Intermediate Ancestor** - CBCA-optimized variant
- **Confidence Score**: 0.92
- **Specialization**: CBCA-specific (85% selectivity)
- **Characteristics**: Key evolutionary intermediate
- **Industrial Use**: Specialized CBCA manufacturing

### HCa.fasta
**Recent Ancestor** - Balanced production profile
- **Confidence Score**: 0.89
- **Specialization**: Promiscuous (40% THCA, 30% CBDA, 30% CBCA)
- **Characteristics**: Modern-like with ancestral promiscuity
- **Industrial Use**: Balanced multi-cannabinoid extraction

## Format

All files follow standard FASTA format:
```
>Header | Metadata
SEQUENCE
```

**Header Components**:
- Enzyme name
- Reconstruction method
- Confidence score
- Specialization notes

## Usage

These sequences can be:
1. Uploaded to the AncestryBio Dash application
2. Used for phylogenetic analysis
3. Referenced in research publications
4. Synthesized for experimental validation

## Source

**Study**: Wageningen University Ancestral Cannabinoid Biosynthesis Study  
**Date**: December 2025  
**Method**: Maximum Likelihood Reconstruction  
**Host**: Saccharomyces cerevisiae CEN.PK113-7D
